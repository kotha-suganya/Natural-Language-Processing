# Natural-Language-Processing

Basic NLP Algorithms & Techniques in Python
This repository contains a collection of Python scripts demonstrating essential Natural Language Processing (NLP) techniques. From basic text preprocessing to machine learning classifiers, these examples serve as a "cheat sheet" for anyone getting started with NLP.
🚀 Features & Modules
1. Text Preprocessing
Lemmatization: Converting words to their dictionary base form (using nltk).
Stemming: Reducing words to their root form (using PorterStemmer).
Stopwords Removal: Filtering out common filler words (e.g., "the", "is", "a").
Tokenization: Breaking sentences into individual words or units.
2. Lexical Analysis
Word Count: Using Counter to find term frequency and most common words.
N-Grams: Generating Bigrams and Trigrams to understand word context.
Minimum Edit Distance: Calculating the number of edits needed to change one word into another.
